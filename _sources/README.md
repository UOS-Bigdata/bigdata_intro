## 학습 목표

빅데이터입문은 최신 프로그래밍 언어를 이용하여 데이터를 수집, 정리, 요약하는 자료 처리에 필요한 기초적인 프로그래밍 역량을 키운다. 또한 데이터를 분석하여 그 특성을 파악하고 합리적인 결론을 유추할 수 있는 통계적 분석 이론과 방법의 기초를 학습한다. 

이 과목을 수강하기위하여 필요한 별도의 선수과목은 없다.
